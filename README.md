# GlobalHookApp

A simple Windows Forms app to demonstrate keyboard/mouse global hook (coming soon).

## How to Run

- Open with Visual Studio 2022 or later
- Compile and run
---
✅ LANGKAH 3: Jalankan Proyek (opsional, pakai Visual Studio)

    Cukup buka folder

    Pastikan target .NET sudah diinstal (NET 6.0+)

    Klik Run
